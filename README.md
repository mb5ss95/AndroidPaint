# PaintApp
## 설 명(Explanation)
- 한글 인식을 이용한 메모장과 그림판 기능이 있습니다.
- 메모장은 .txt로 저장하고, 그림판은 .jpeg으로 저장됩니다.
  
## 기 능(Function)
- 한글 인식
- 메모장
- 그림판
